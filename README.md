# Only-test-task
[Тестовое задание от Only.](https://ribreak.github.io/Only-test/)

## Запуск

1. Clone the repository:
   
```bash
git clone https://github.com/Ribreak/only-test.git
```

2. Navigate to the project directory:

```bash
cd only_test
```

3. Install dependencies:

```bash
npm install
```

4. Start app

```bash
npm start
```
